# no-clickable-div

Flags `<div>` or `<span>` with `(click)`, `(keydown)`, or similar activation bindings. Prefer `<button type="button">` or `<a>` (with `href` / `routerLink`) so assistive tech and keyboard users get correct semantics and behavior.

## Rule details

This rule is a complement to `@angular-eslint/template/interactive-supports-focus` and related accessibility rules.
